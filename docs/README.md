# 学习笔记（docs/）

从零手搓 ContextForge 这一路，把学到的**认知性经验**沉淀在这里——给人看，也给 AI 看。

不是运行记录，也不是排错手册。每篇讲透一个「我学到了什么」，结构统一：

> **问题是什么 → 我原来怎么想 → 真相 / 原理 → 学到的道理**

标题一眼见结论。挑你感兴趣的看，不必按顺序。

- [01 · KV Cache：每轮全量历史为何不贵](./01-kv-cache-每轮全量历史为何不贵.md) —— `in=2` 的真相：input 只是未缓存的新增量，其余命中缓存

- [02 · 上下文压缩不是截断 API，而是本地重写历史](./02-上下文压缩不是截断API而是本地重写历史.md) —— API 无状态，压缩 = 我们在本地重写 messages

- [08 · 循环 import 是「东西放错了层」的信号](./08-循环import是东西放错了层的信号.md) —— 延迟 import 是治标，根治是挪到依赖正确的层
